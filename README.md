## sfit-api-server

### Prerequisites

- Node.js 18+
- PostgreSQL instance and connection string

### Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy `.env.example` to `.env` and set your values.
3. Start the server:
   ```bash
   npm run dev
   ```

### Routes

- `GET /api/health` &mdash; basic health check.
